# API Description
1
 PUT /services/{serviceName}
Consume: Multipart/Form-data		 Submit and deploy a service*. Tosca is embedded in a HTML form.
2
 PUT /services/xml
Consume:Application/ XML		 Submit and deploy a service*. Tosca is embedded in a HTML form.
3
 GET /services/{serviceId}
Produce: Application/XML		 Get the deployment structure of the application.
4
 POST /services/{serviceId}/redeploy Redeploy the service
5
 DELETE /services/{serviceId}
 Remove the service
6
 POST /services/{serviceId}/nodes/{nodeId}/instance-count/{quantity}
 Deploy more units.
7
 POST /services/{serviceId}/nodes/{nodeId}/instances/{instanceId}/action_queue/{actionName}
 Execute a reconfiguration. The command is put in to a queue.
8
 DELETE /services/{serviceId}/nodes/{nodeId}/instances/{instanceId}
 Delete a specific instance by ID.
9 GET /viewgenerator/cloudservice/json/compact/
 Show application structure as a tree.
10
 GET /viewgenerator/cloudservice/json/list
Produce: Application/JSON		 Get the list of the applications.
11
 POST /elise/communication/queryUnitInstance
Produce: Application/JSON		 Start a query from external service for the configuration information.
12
 GET /elise/manager/query/{queryUUID}
Produce: Text/plain		 Check the status of the information collection process of (11)
13
 GET /elise/unitinstance/{instanceUUID}
Produce: Application/JSON		 Get the more information of an instance after (12). The instanceUUID is get from (3) or (9).
