Introduction

This page provides extra materials for the paper "Managing Information for Dynamic Configuration of Elastic IoT Cloud Systems".

At the moment, ELISE is integrated in SALSA, towards providing information of IoT Cloud Systems for the configuration processes. By the integration, ELISE reuses some of concepts and designs in SALSA, as well as automatically captures deployment information from SALSA by default, such as the cloud service structure and cloud infrastructure information.

ELISE API

API for managing information

In current version, the information of provider and cloud service structure is static collected, thus it can be query dirrectly in ELISE. The unit instances are dynamically create/remove and being distributed over infrastructures, thus the collection process can take time. To deal with this, ELISE provides a separate API to start the queries and an API to get the information.

# APIs Description
1 GET /provider Get list of providers
2 GET /provider/{uniqueID} Get list of offered service from a provider
4 GET /cloudservice Get list of cloud services
3 GET /cloudservice/{uniqueID} Get cloud service's structure
4 POST /communication/queryUnitInstance Start a query by forwarding request to collectors. This consume a query string and return an ID of the query to check
5 GET /communication/query/{queryUUID} Return status of a query: how many collectors are involving to get information for that query, what is the status of the collection process
6 POST /unitinstance/{uniqueID} Get details the information of single instance

API for managing ELISE components

# APIs Description
1 GET /manager/count Get the description of running collectors
2 POST /manager/conductor/salsa/{pioneerID} Request SALSA to deploy a conductor (which host collector artifact)
3 POST /manager/conductor/{conductorID}/collector/{collectorName} Push a collector artifact to a conductor
4 GET /manager/collector Get the list of supported collectors which can be deployed
5 POST /manager/conductor/resyn Synchronize the collectors with elise master
