The steps to setup SALSA are:
1. SALSA is packed as web application, running well on Tomcat server.
2. Configure the salsa-engine parameters in src/resources
3. Compile the main project.
4. Deploy salsa-engine.war to the Tomcat
5. Push salsa-pioneer.jar to a repository to have the direct URL to download it. Note: the URL is configured in step 2 too.
6. Access to the salsa-engine application to get the UI at: http://<IP of the machine>:8080/salsa-engine
7. Using RESTful API at http://<IP of the machine>:8080/salsa-engine/rest
