SALSA 2.0

SALSA 2.0 improve the communication via message queue and provide rich repository API. To run SALSA you need to put salsa-engine.jar and salsa-pioneer.jar in the same folder and run "java -jar salsa-engine.jar" (see the documentation for more details). Salsa client can be used as command line or imported into your project to interact with SALSA.

Source: zip, View on github

Release:

  • Salsa Engine: salsa-engine.jar
  • Salsa Pioneer:salsa-pioneer.jar
  • Salsa Client:salsa-client.jar
