SALSA 2.0

SALSA 2.0 improves the communication via message queue and provides rich repository APIs. Salsa client can be used as commandline or to import into Java projects to access Java API. The following URLs are for production branch, which contains only stable features. The master branch contains many dependencies in our private repository and just for development process.

Source: zip, View on github

Release:

  • Salsa Engine: salsa-engine.jar
  • Salsa Pioneer:salsa-pioneer.jar
  • Salsa Client:salsa-client.jar

The service can be run by following command: $ java -jar salsa-engine.jar. Please see the documentation for more details.

By default, following endpoints can be accessed:

  1. Web interface: http://localhost:8080/salsa-engine
  2. SALSA APIs: http://localhost:8080/salsa-engine/rest
  3. ELISE APIs: http://localhost:8080/salsa-engine/rest/elise
